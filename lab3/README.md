### 4.*Оформление заявки на покупку акций (рыночная)*

| Название                                                     | Описание                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Идентификатор**                                            | 4                                                            |

| Потворно нажать на кнопку "отправить заявку".				   | Появленя уведомление об отправки заявки,а также появление этой заявки в блоке заявок. |
